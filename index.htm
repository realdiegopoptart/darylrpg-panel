Welcome

Z-WAMP Server Pack is a lightweight zero-install Web server package that runs on Windows. The project aims to provide the latest production/stable versions of Apache, MySQL, PHP, Adminer, MongoDB Admin, MemCached, SQLite, eAccelerator, and Alternative PHP Cache (APC) to boost server performance.

You can download the latest release of Z-WAMP at:-

http://sourceforge.net/projects/zwamp

If you feel that the Z-WAMP Server Pack is one great addition to your Web development arsenal and it saves you a lot of time and money, please consider making a donation to the project. Aside from motivating the people who make this package available to the world, it also helps keep the project alive and up-to-date.
